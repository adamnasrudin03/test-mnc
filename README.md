## Development Guide

### Collection Using Postman
- ./go-test-mnc.postman_collection.json
  
### Installation
- Clone this repo

    ```sh
        git clone https://github.com/adamnasrudin03/go-test-mnc.git
    ```

- Copy `.env.example` to `.env`

    ```sh
        cp .env.example .env
    ```
- Setup local database
- Start service API
    ```sh
        go run main.go
    ```

## Coverage Unit Test
